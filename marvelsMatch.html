Instructions

  • Please set the timer before strating the game, if timer is not set then the game will not start.
  • To set the timer please select any timer value i.e 30, 60 or 90.
  • 90--> Easy Level
  • 60--> Medium Level
  • 30--> Hard Level
  • You need to match all the cards wuth their pairs to get points.
  • To match, tap the cards and cards will get flipped and shows the character. Keep the shown character position in the memory and flip the cards again and find the match.
  • Complete the task within the selected time to win!
